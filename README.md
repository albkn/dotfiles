# dotfiles
Simple config dotfiles for bash, vim, etc

## How to use
1. Go to home directory  
    
    ```shell
    $ cd ~
    ```
    
2. Clone from the repo to `.dotfiles` diretory  

    ```shell
    $ git clone https://github.com/morcerf/dotfiles .dotfiles
    ```
    
3. Go into the directory and run the script

    ```shell
    $ cd ~/.dotfiles
    $ ./run.sh link
    ```

And you are done :D
